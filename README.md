本仓库用于存档开源文档及其加工产物

**仅用于学习用途，不保证及时更新**

## 加工规范

- `*.md` 原始文档
- `*.fmt.md` （如有必要）格式化原始文档
- `*.ipynb` 笔记文档（将 `(j|t)sx?` 代码块转换为 `code cell - Deno Kernel`）
- `*.zh.md` 中文文档
- `*.zh.ipynb` 中文笔记文档
